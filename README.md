## photo_board
2019년 멋쟁이사자처럼 동아리 활동을 하면서 Html/css 그리고 Django를 한 학기동안 배우고 혼자 학습 + 응용한 프로젝트입니다!
기본적으로 게시글 CRUD, 댓글을 구현하고 좋아요, 조회수를 추가 구현하였습니다.
<br>

> **install** 
 - pip install django
 - pip install django-imagekit
 - pip install pillow
 - pip install django-hitcount
 - pip3 install whitenoise
<br><br><br>
